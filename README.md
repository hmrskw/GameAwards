# Magnolia  
  
## 作品名  
  Magnolia

## 概要  
  この作品は日本ゲーム大賞2017アマチュア部門に向けてチームで制作したゲームです。  
  1つのコントローラーを使って2体のキャラクターを操作し、マップを探索するゲームです。
  
## プラットフォーム  
  Windows8,Windows8.1,Windows10  
  (Xbox360コントローラーが必要です。)  
  
## 使用ツール
  * Unity5.6.0f3
  * VisualStudio2015
  * Maya
  * MotionBuilder
  * SAI
  * PhotoShop CC
